# 🍽️ Dishcovery

Dishcovery is a **recipe discovery web application** that helps users find dishes based on ingredients, categories, and random suggestions.
The goal of the project is to make it easier for people to decide **what to cook using the ingredients they already have**.

With a clean interface and simple navigation, Dishcovery allows users to quickly explore recipes and discover new dishes.

---

## ✨ Features

### 🔎 Search Recipes by Ingredient

Users can search for recipes using ingredients they already have at home.

### 🍰 Browse Recipes by Category

Explore dishes grouped into different categories such as desserts, snacks, and main courses.

### 🎲 Random Recipe Generator

A “Random Recipe” option suggests a dish when the user wants inspiration.

### 🎨 Clean and Responsive UI

The application uses **Tailwind CSS** to create a responsive and modern user interface.

## 🚀 Getting Started

Follow these steps to run the project locally.

### 1. Clone the repository

```
git clone https://github.com/KeerthiSaranya17/Dishcovery.git
```

### 2. Navigate to the project folder

```
cd Dishcovery
```

### 3. Install dependencies

```
npm install
```

### 4. Start the development server

```
npm run dev
```

The application will run at:

```
http://localhost:5173
```
